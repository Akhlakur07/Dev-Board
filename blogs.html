Question-1: What are the different ways to select an element in the DOM?

1.By ID, 2. By class name, 3. By tag name, 4. By attribute, 5. By CSS selector, 6. By querySelector.

Question-2: What is the difference between innerHTML, innerText, and textContent

1. innerHTML: Sets or gets HTML content, 2. innerText: Sets or gets visible text, 3. textContent: Sets or gets all text content, including hidden elements.

Question-3: What is event delegation in the DOM?

Attach event listeners to a parent element, 2. Parent handles events for dynamically added child elements via event bubbling.

Question-4: What is event bubbling in the DOM?

Event triggered on child element, 2. Propagates upward to parent elements in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create: document.createElement(), 2. Add: appendChild(), append(), insertBefore(), 3. Remove: remove(), removeChild().
